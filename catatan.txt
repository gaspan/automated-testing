### macam-macam test case
    1. positive test case
        yaitu mengetes API dengan mindset consumer(fe/android) yg taat pada standar 

    2. negative test case
        yaitu mengetes API dengan mindset user yg tidak sesuai standar
        contoh nya: salah satu atau beberapa column request di hilangkan / disalahkan / dikosongkan, 
        value nya tidak sesuai dengan standar
